 16    """ 
 17    The Mobot Class 
 18    =============== 
 19   
 20    Each instance of this class can be used to represent a physical Mobot. The 
 21    member functions of this class can be used to get data, set motor angles, 
 22    scan for button events, and more.   
 23   
 24    Blocking and Non-Blocking Member Functions 
 25    ========================================== 
 26   
 27    The member functions of the Mobot class which are responsible for moving 
 28    the joints of the Mobot can be categorized into two general types of 
 29    functions; "blocking" functions and "non-blocking" functions. A blocking 
 30    function is a function that "hangs" until the complete motion is done, 
 31    whereas a "non-blocking" function returns as soon as the motion begins, 
 32    but does not wait until it is done. In the Mobot class, all functions 
 33    are blocking unless the have the suffix "NB", such as "Mobot.moveNB()". 
 34   
 35    For example, consider the following lines of code:: 
 36      mobot.move(360, 0, 0, 360) 
 37      angles = mobot.getJointAngles() 
 38    When the above lines of code are executed, the Mobot will rotate joint 1 
 39    360 degrees. Once the joint has rotated the full revolution, the joint 
 40    angles will be retrieved and stored in the variable "angles". Now  
 41    consider the following code:: 
 42      mobot.moveNB(360, 0, 0) 
 43      angles = mobot.getJointAngles() 
 44    For these lines of code, joint 1 also moves 360 degrees. The difference is  
 45    that with these lines of code, the joint angles will be retrieved as soon 
 46    as the joint begins turning, instead of waiting for the motion to finish. 
 47    This is because the non-blocking version of move() was used, which returns 
 48    immediately after the joint begins moving allowing the getJointAngles()  
 49    function to execute as the joint begins moving. 
 50   
 51    The L{moveWait()<barobo.mobot.Mobot.moveWait>} function can be used to 
 52    block until non-blocking motion functions are finished. For instance, the 
 53    following two blocks of code will accomplish the same task:: 
 54      mobot.move(360, 0, 0) 
 55      angles = mobot.getJointAngles() 
 56   
 57      mobot.moveNB(360, 0, 0) 
 58      mobot.moveWait() 
 59      angles = mobot.getJointAngles() 
 60   
 61    """
