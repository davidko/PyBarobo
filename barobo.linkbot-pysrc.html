 17    """ 
 18    The Linkbot Class 
 19    ================= 
 20   
 21    Each instance of this class can be used to represent a physical Linkbot. The 
 22    member functions of this class can be used to get data, set motor angles, 
 23    beep the buzzer, scan for button events, and more.   
 24   
 25    Blocking and Non-Blocking Member Functions 
 26    ========================================== 
 27   
 28    The member functions of the Linkbot class which are responsible for moving 
 29    the joints of the Linkbot can be categorized into two general types of 
 30    functions; "blocking" functions and "non-blocking" functions. A blocking 
 31    function is a function that "hangs" until the complete motion is done, 
 32    whereas a "non-blocking" function returns as soon as the motion begins, 
 33    but does not wait until it is done. In the Linkbot class, all functions 
 34    are blocking unless the have the suffix "NB", such as "Linkbot.moveNB()". 
 35   
 36    For example, consider the following lines of code:: 
 37      linkbot.move(360, 0, 0) 
 38      linkbot.setBuzzerFrequency(440) 
 39    When the above lines of code are executed, the Linkbot will rotate joint 1 
 40    360 degrees. Once the joint has rotated the full revolution, the buzzer will 
 41    sound. Now consider the following code:: 
 42      linkbot.moveNB(360, 0, 0) 
 43      linkbot.setBuzzerFrequency(440) 
 44    For these lines of code, joint 1 also moves 360 degrees. The difference is  
 45    that with these lines of code, the buzzer will begin emitting a tone as soon 
 46    as the joint begins turning, instead of waiting for the motion to finish. 
 47    This is because the non-blocking version of move() was used, which returns 
 48    immediately after the joint begins moving allowing the setBuzzerFrequency()  
 49    function to execute as the joint begins moving. 
 50   
 51    The L{moveWait()<barobo.linkbot.Linkbot.moveWait>} function can be used to 
 52    block until non-blocking motion functions are finished. For instance, the 
 53    following two blocks of code will accomplish the same task:: 
 54      linkbot.move(360, 0, 0) 
 55      linkbot.setBuzzerFrequency(440) 
 56   
 57      linkbot.moveNB(360, 0, 0) 
 58      linkbot.moveWait() 
 59      linkbot.setBuzzerFrequency(440) 
 60   
 61    """
